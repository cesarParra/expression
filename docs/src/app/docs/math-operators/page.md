- `+` Addition

```apex
expression.Evaluator.run('1 + 1'); // 2
```

- `-` Subtraction

```apex
expression.Evaluator.run('1 - 1'); // 0
```

- `*` Multiplication

```apex
expression.Evaluator.run('2 * 2'); // 4
```

- `/` Division

```apex
expression.Evaluator.run('4 / 2'); // 2
```

- `^` Exponentiation

```apex
expression.Evaluator.run('2 ^ 2'); // 4
```
